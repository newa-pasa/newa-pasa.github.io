Scrum Poker

A real-time Scrum Poker application to help agile teams estimate tickets.

Created by:

Dilip Shrestha K.

Completed on:

May 2025

Skills:

React / WebSockets / Java

This application allows agile teams to estimate tickets effectively in real-time. Key features include the ability to create and join game rooms, with live updates as participants join and cast their votes. A pie chart provides a clear visual representation of the voting results, facilitating quick consensus.

It is designed to streamline the estimation process in agile development, making sprint planning more efficient and collaborative.
